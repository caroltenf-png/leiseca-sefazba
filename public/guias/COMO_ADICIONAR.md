# Como adicionar seus guias HTML

Coloque cada arquivo HTML nesta pasta. O nome do arquivo define o ID da disciplina.

## Convenção de nomes:
- dir-tributario.html         → Direito Tributário
- dir-administrativo.html     → Direito Administrativo
- dir-constitucional.html     → Direito Constitucional
- contabilidade-geral.html    → Contabilidade Geral
- legislacao-ba.html          → Legislação Tributária BA
- auditoria.html              → Auditoria
- estatistica.html            → Estatística
- portugues.html              → Português
- informatica.html            → Noções de Informática
- raciocinio-logico.html      → Raciocínio Lógico
- igualdade-racial.html       → Igualdade Racial e Gênero
- dir-financeiro.html         → Direito Financeiro

## Como funciona:
A plataforma carrega automaticamente qualquer .html que você colocar aqui.
O arquivo aparece na tela Guias com ícone, nome e disciplina corretos.
